# Security Policy

## Supported Versions

| Version | Supported                         |
| ------- | --------------------------------- |
| 6.0.x   | :white_check_mark: Active support |
| 5.7.x   | :wrench: Security fixes only      |
| < 5.7   | :x: End of life                   |

## Reporting a Vulnerability

To disclose a security vulnerability, email [lti-1p3-tool@packback.co](mailto:lti-1p3-tool@packback.co).
